## ifconfig

**ifconfig** 是我们最常用的查看当前系统的网卡和 ip 地址信息的命令。如果你的机器上还没安装 **ifconfig** 命令，可以使用如下命令安装：

```
yum install net-tools
```



## ping

**ping** 命令是我们最常用的命令之一，一般用于侦测本机到目标主机的网络是否通畅。

> ping 命令是通过发送 ICMP 数据包来实现的。
